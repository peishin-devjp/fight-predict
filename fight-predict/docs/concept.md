# Fight Predict

## コンセプト
Fight Predictは、格闘技ファンが予想を楽しみ、その知識や分析を競い合い、同じ趣味を持つ仲間と交流できるコミュニティを目指す。
「予想・競争・交流」の3本柱

## MVP
RIZIN大会の勝敗予想とオッズ考慮ランキング

## ランキング方式
オッズ考慮ポイント制
